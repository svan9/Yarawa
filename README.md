## Алфавит и произношение

---

### Спецификация



| Буква |         Случай          |     Звук      |
| :---: | :---------------------: | :-----------: |
|  `A`  | если не стоит последней |     `аэ`      |
|   ↳   |  если стоит последней   |     `а.`      |
|  `B`  |     после `E`, `I`      |      `в`      |
|   ↳   |       в отсальных       |      `б`      |
|  `I`  |            -            |      `ы`      |
|   ↳   |        во начале        |      `и`      |
|  `E`  |            -            |      `Э`      |
|  `~`  |       после букв        | смягчает звук |
|  `w`  |            -            |      `ф`      |
|  `q`  |            -            |      `у`      |
|  `u`  |            -            |      `Ϋ`      |
