## Алфавит и произношение

---

### Спецификация



| Буква |         Случай          |     Звук      |
| :---: | :---------------------: | :-----------: |
|  `A`  | если не стоит последней |     `аэ`      |
|   ↳   |  в конце или после `T`  |   `а.`/ `а`   |
|  `B`  |     после `E`, `I`      |      `в`      |
|   ↳   |       в отсальных       |      `б`      |
|  `I`  |            -            |      `ы`      |
|   ↳   |        во начале        |      `и`      |
|  `E`  |            -            |      `Э`      |
|  `~`  |       после букв        | смягчает звук |
|  `w`  |            -            |      `ф`      |
|  `q`  |            -            |      `у`      |
|  `u`  |            -            |      `Ϋ`      |
|  `v`  |         перед `y`       |      `и`      |
|  `v`  |         в конце         |       -       |
|  `y`  |         в конце         |   -/`ы...`    |
|  `vy` |            -            |    `и`/`ы`    |


Удвоенние буквы дает мягкий звук, также `~` сверху буквы тоже дает мягкость звука.
